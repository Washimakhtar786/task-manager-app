# 📝 Task Manager Application

A full-stack Task Manager application built using **React, Node.js, Express.js, PostgreSQL, and Sequelize**.

The application allows users to register, log in, manage their personal tasks, and provides an Admin Dashboard for managing users and monitoring the system.

---

# 🚀 Features

## Authentication

- User Registration
- User Login
- JWT Authentication
- Logout
- Protected Routes
- Role-Based Access Control

## Task Management

- Create Task
- View Tasks
- Update Task
- Delete Task
- Task Status Filter
- Task Summary
- Success & Error Messages
- Confirmation Modal

## Admin Dashboard

- Dashboard Statistics
- View All Users
- Activate / Deactivate Users
- View All Tasks
- Role-Based Authorization

# 📦 Installation

Clone the repository

```bash
git clone <your-repository-url>
```

Move into project

```bash
cd task-manager-app
```

---

## Backend Setup

```bash
cd server

npm install
```

Run backend

```bash
npm run dev
```

---

## Frontend Setup

```bash
cd frontend

npm install
```

Run frontend

```bash
npm run dev
```
